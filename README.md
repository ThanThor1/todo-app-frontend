# Todo App Frontend

Giao diện người dùng cho ứng dụng quản lý công việc.

## Công nghệ sử dụng
- JavaScript
- React (Vite)

## Tính năng
- Đăng ký, đăng nhập
- Quản lý công việc theo trạng thái: Todo, In Progress, Done
- Kanban board 3 cột

## Cài đặt
1. Clone repo
2. Chạy `npm install`
3. Chạy `npm run dev`
